# Inventory Management System

## Overview
This is a Tkinter GUI application designed for managing inventory efficiently.

## Entry Point
The application can be started from the `IMS/login.py` file.

## Database
It uses the included SQLite database file located at `IMS/ims.db`.

## Features
- User-friendly interface
- Real-time inventory tracking
- Easy to use navigation

Please refer to the additional sections in this README for more information on usage and features.
